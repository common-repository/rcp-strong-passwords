Enforce strong passwords during user registration for Restrict Content Pro

== Description ==

**On October 14th, 2021, all Restrict Content Pro add-ons will be removed from the WordPress plugin repository.**

**This plugin and all other Restrict Content Pro add-ons will remain available to download in your <a href="https://members.ithemes.com/panel/downloads.php">iThemes Member's Panel</a>.**

This is a simple extension for [Restrict Content Pro](https://restrictcontentpro.com/) to enforce strong passwords during registration. This extension does NOT function without Restrict Content Pro installed.

This extension has no options and will automatically require users to use a strong password during registration.

Learn more about Restrict Content Pro: https://restrictcontentpro.com/
